# Σύστημα Διαχείρισης Εισιτηρίων Κινηματογράφου (Cinema Ticket System)

Μια πλήρης full-stack εφαρμογή για διαχείριση εισιτηρίων σινεμά, κατασκευασμένη με **Laravel (Backend)** και **React (Frontend)**.

## 🚀 Οδηγίες Εγκατάστασης & Εκτέλεσης

Η εφαρμογή αποτελείται από δύο μέρη: το Backend (API & Βάση Δεδομένων) και το Frontend (Διεπαφή Χρήστη). Θα χρειαστείτε **δύο τερματικά** ανοιχτά ταυτόχρονα.

### Προαπαιτούμενα
- PHP & Composer
- Node.js & NPM
- MySQL

---

### 1️⃣ Εκκίνηση Backend (Laravel)

Το backend διαχειρίζεται τη βάση δεδομένων και παρέχει το API.

1. Ανοίξτε ένα τερματικό και πηγαίνετε στον φάκελο `backend`:
   ```bash
   cd backend
   ```

2. Εγκαταστήστε τις βιβλιοθήκες PHP (αν δεν το έχετε κάνει ήδη):
   ```bash
   composer install
   ```

3. Ρυθμίστε το περιβάλλον:
   - Αντιγράψτε το `.env.example` σε `.env`: `cp .env.example .env`
   - Ρυθμίστε τα στοιχεία της βάσης δεδομένων στο αρχείο `.env` (DB_DATABASE, DB_USERNAME, DB_PASSWORD).

4. Δημιουργήστε το κλειδί της εφαρμογής και τη βάση δεδομένων:
   ```bash
   php artisan key:generate
   php artisan migrate:fresh --seed
   ```
   *(Η εντολή `--seed` γεμίζει τη βάση με ταινίες, αίθουσες και προβολές)*.

5. Ξεκινήστε τον server:
   ```bash
   php artisan serve
   ```
   *Θα τρέχει στο: http://localhost:8000*

---

### 2️⃣ Εκκίνηση Frontend (React)

Το frontend είναι η ιστοσελίδα που βλέπει ο χρήστης.

1. Ανοίξτε ένα **νέο** τερματικό (κρατήστε το backend ανοιχτό!).
2. Πηγαίνετε στον φάκελο `frontend`:
   ```bash
   cd frontend
   ```

3. Εγκαταστήστε τις βιβλιοθήκες JavaScript (αν δεν το έχετε κάνει ήδη):
   ```bash
   npm install
   ```

4. Ξεκινήστε την εφαρμογή:
   ```bash
   npm run dev
   ```
   *Θα τρέχει στο: http://localhost:5173* (ανοίξτε το στον browser).

---

## 🔑 Χρήση της Εφαρμογής

### Ρόλοι Χρηστών

1. **Απλός Χρήστης (User)**
   - Μπορεί να δει ταινίες και προβολές.
   - Μπορεί να κάνει κράτηση εισιτηρίων.
   - Βλέπει το ιστορικό κρατήσεών του στο Προφίλ.
   - **Εγγραφή**: Πατήστε "Sign Up" στην αρχική ή πηγαίνετε στο `/register`.

2. **Διαχειριστής (Admin)**
   - Έχει πρόσβαση στο **Admin Dashboard**.
   - Μπορεί να δει όλες τις κρατήσεις, τα ονόματα των πελατών και τις αίθουσες.
   - **Εγγραφή Διαχειριστή**: Πηγαίνετε στη σελίδα `/admin/register` για να δημιουργήσετε λογαριασμό διαχειριστή.
   - *Σημείωση: Μόνο οι χρήστες που εγγράφονται από αυτή τη σελίδα έχουν δικαιώματα διαχειριστή.*

## 🛠 Τεχνολογίες

- **Backend**: Laravel 10+, MySQL, Sanctum (Authentication).
- **Frontend**: React 18+, Tailwind CSS, Axios, React Router.
